JSP Introduction Quiz

Q1. What is the main value of Java Server Pages?

A1. JavaServer Pages(JSP) is a complimentary technology to Java Servlet for developing Web pages that supports dynamic content.It helps us to insert java code in HTML pages by making use of special JSP tags.

Q2. How are JSP pages related to servlets?

A2. JSPs are always compiled in to servlets.

Q3. How are JSP pages related to HTML?

A3.JSP are actually an HTML file that allows dynamic content by embedding java programming.

Q4. Give an example of a JSP scriptlet and show how it will look in the JSP servlet.

A4. A scriptlet tag is used to execute java source code in JSP and inside JSP it looks like this <% int i=6; int k=i+10; %>.

Q5. Give an example of a JSP declaration and show how it will look in the JSP servlet.

A5. We can declare static member, instance variable and methods inside Declaration Tag and inside JSP it look like this <%! int count=0; %>.

Q6. Give an example of a JSP comment and show how it will look in the JSP servlet.

A6. Inside JSP a comment looks like this <%-- This is a JSP comment --%>.

Q7. Give an example of a JSP expression and show how it will look in the JSP servlet.

A7. Expression Tag is used to print out java language expression that is put between the tags and inside JSP it looks like this <%= (2*5) %>.

Q8. Give an example of a JSP directive and show how it will look in the JSP servlet.

A8. Directive Tag gives special instruction to Web Container at the time of page translation and inside JSP it looks like this <%@ page import="java.util.Date" %> or <%@ include file="label.jsp" %> or <%@ taglib prefix='c' uri='http://java.sun.com/jsp/jstl/core' %>.

Q9. Explain how an EL expression is evaluated.

A9. EL can be evaluated in two ways, i.e as attribute values in standard and custom tags and as output in HTML tag.

Q10. Explain how servlet attributes are involved in EL expressions.

A10. If our servlet set the attributes in any scope (request, session etc), the attribute can be accessed inside the JSP by the name only without mentioning the scopes (scopes are implicit for the JSP) .

Q11. Explain how servlets and JSPs use request dispatch to interact.

A11. Servlets uses the JSP as a view to implement the MVC architecture and the RequestDispatcher object of the servlet can forward the request to the JSP by mentioning the JSP name only (name.jsp).
