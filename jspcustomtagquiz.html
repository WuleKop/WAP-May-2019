JSP Custom Tags Quiz

Q1. Rewrite the JSP page in the first demo (forEach) to use scripting instead of the JSTL forEach.

A1. JavaServer Pages(JSP) is a complimentary technology to Java Servlet for developing Web pages that supports dynamic content.It helps us to insert java code in HTML pages by making use of special JSP tags.

Q2. How do custom tags relate to JSTL?

A2. JSPs are always compiled in to servlets.

Q3. What is the role of the URI in the TLD and the taglib directive?

A3.The URI in the taglib directive specifies where to find the tag library description file.

Q4. What is a tag handler class?

A4. A tag handler class is a class that extend SimpleTagSupport class and override the doTag() method, where we can place our code to generate content for the custom tag.

Q5. What is the role of attribute setters in a tag handler class?

A5. We can declare static member, instance variable and methods inside Declaration Tag and inside JSP it look like this <%! int count=0; %>.

Q6. What is the role of the doTag() method in a tag handler class?

A6. Inside JSP a comment looks like this <%-- This is a JSP comment --%>.

Q7. What does the operation getJspContext().getOut().write(“Hi Bob”) do when called in a doTag() method?

A7. Expression Tag is used to print out java language expression that is put between the tags and inside JSP it looks like this <%= (2*5) %>.

Q8. What does the operation getJspBody().invoke(null) do when called in a doTag() method?

A8. Directive Tag gives special instruction to Web Container at the time of page translation and inside JSP it looks like this <%@ page import="java.util.Date" %> or <%@ include file="label.jsp" %> or <%@ taglib prefix='c' uri='http://java.sun.com/jsp/jstl/core' %>.
