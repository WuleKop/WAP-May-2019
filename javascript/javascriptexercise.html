Q1. What is the difference between the following 2 statements?

A1. The second one calls the function booyah immediately rather than wait 2000ms

Q4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

A4. Unobstructive Javascript is a Javascript code that is not embedded (separated) from the HTML code. The Javascript code can only be attached at the head of the HTML by using the script tag.
Its main application is to facilitate progressive enhancement and separation of functionality from content/presentation.
